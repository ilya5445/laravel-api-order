
# Laravel API

## Методы API

- Получение списка заказов GET /api/orders/
- Получение конкретного заказа GET /api/orders/{id}
- Создание заказа POST /api/orders/
- Обновление заказа PUT /api/orders/{id}
- Удаление заказа DELETE /api/orders/{id}
